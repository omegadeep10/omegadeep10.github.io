SWORDS Redesign

In this school project, I led a team in redesigning the user interface of SWORDS, which is a student course registration management system (among other things). We also showcased our project using Invision, which is a tool for simulating interactivity.

Unbeatable Tic-Tac-Toe

Over spring break, I created a simple command-line based tic-tac-toe game in Python. This game uses a recursive minimax algorithm to determine the best possible move for a given board state. I created this in a few days during my spring break after being inspired by Google’s AlphaGo Project.

JavaScript Calculator

I made a simple calculator to learn basic JavaScript. The calculator has a history feature that remembers your last calculation. It has all the expected functions of a calculator, such as addition, multiplication, subtraction, and division. The app features keyboard shortcuts so users do not have to click buttons repeatedly.
